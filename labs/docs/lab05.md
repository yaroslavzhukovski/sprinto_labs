# Lab 05 - Monitoring and Alerts

## Scenario

Diagnostic settings are disabled in Terraform for this run. Logs and metrics are incomplete, and expected alerts do not trigger.

## Learning objectives

- Configure diagnostics to Log Analytics.
- Query telemetry availability.
- Validate metric alert logic and action groups.

## Student tasks

1. Confirm missing diagnostics on VM/storage/key vault resources.
2. Re-enable diagnostics and validate data ingestion.
3. Trigger a simple alert condition (for example CPU load).
4. Confirm alert fired and action group processed notification.

## Success criteria

- Diagnostics are enabled on required resources.
- Alert transitions to fired state with evidence.

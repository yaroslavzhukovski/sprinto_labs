# Lab 06 - Bastion and VM Scale Set

## Scenario

Bastion and a small VMSS are enabled. Students must validate secure access and basic scale behavior while staying within budget guardrails.

## Learning objectives

- Use Bastion for browser-based VM access.
- Inspect VMSS instance health and model settings.
- Practice safe scale actions and rollback.

## Student tasks

1. Verify Bastion deployment prerequisites and host status.
2. Connect to VM without opening inbound SSH on Internet.
3. Increase VMSS instance count to 2 and observe provisioning.
4. Scale back down to 1 and verify cost-conscious operation.

## Success criteria

- Bastion access works for target VM.
- VMSS scale out/in completes successfully.
